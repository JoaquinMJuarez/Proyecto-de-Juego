# 🎮 Piedra, Papel o Tijera

Un juego clásico desarrollado con **HTML, CSS y JavaScript** donde el usuario juega contra la computadora.

---

## 📌 Descripción

Este proyecto consiste en un juego interactivo de **Piedra, Papel o Tijera**. El jugador elige una opción y la computadora selecciona otra de forma aleatoria. El sistema determina el ganador y lleva un marcador de la partida.

---

## ✨ Características

- 🪨 Elegir Piedra.
- 📄 Elegir Papel.
- ✂️ Elegir Tijera.
- 🤖 La computadora elige una opción aleatoria.
- 🏆 Determina quién gana cada ronda.
- 📊 Contador de puntos.
- 🔄 Botón para reiniciar la partida.
- 💻 Interfaz sencilla y fácil de usar.

---

## 🎮 Reglas del juego

- Piedra vence a Tijera.
- Tijera vence a Papel.
- Papel vence a Piedra.
- Si ambos eligen la misma opción, es un empate.
